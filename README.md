# Jenat Milan Portfolio

A modern, cyberpunk-themed portfolio website showcasing my work as a Computer Science undergraduate and Full Stack Developer.

## 🚀 Features

- Custom animated splash screen with geometric patterns
- Interactive red dot background animation
- Responsive design with custom cursor effects
- Sections: About, Skills, Projects, Experience, Contact
- AJAX contact form integration with Formspree
- Smooth scroll animations and reveal effects

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom animations, gradients, clip-path)
- Vanilla JavaScript
- Canvas API for animations
- Formspree for contact form

## 📱 Projects Showcased

1. **AI-Based Pet First Aid Chatbot** - React Native mobile app
2. **AI-Powered Website Audit Tool** - React + Node.js web application
3. **Guru AI - Homework Assistant** - Upcoming AI education app

## 🎨 Design Theme

Red and black cyberpunk aesthetic with:
- Orbitron & Rajdhani fonts
- Glowing effects and custom cursor
- Geometric patterns and animations
- Responsive layout
